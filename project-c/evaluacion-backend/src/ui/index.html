Our story

Mesa Larga began in 2019 as a Sunday-night supper club between friends. We kept the long table when we opened the restaurant — eight strangers, one meal, no menus to choose from. It still feels like the right way to eat.

For developers

GET /api/v1/tables/types/
  → [{ id, name, seats, description }]

GET /api/v1/reservations/availability/
  ?date=YYYY-MM-DD&time=HH:MM&party=
  &table_type=&tz=
  → [{ time, table_type, table_type_name,
        seats, available_seats, price_per_seat }]

GET /api/v1/menu/?date=YYYY-MM-DD
  → [{ id, course, name, description,
        price, allergens: [str] }]

GET /api/v1/healthz   → { status: "ok" }
